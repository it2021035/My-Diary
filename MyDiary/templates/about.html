ℹ️ About This Project

My Diary may look like a personal blogging platform — but it’s not what it seems.

This application was intentionally designed as part of an academic thesis project focused on Intrusion Detection Research. The goal is to study how malicious users interact with seemingly legitimate web applications and to collect real-world data for testing defensive cybersecurity mechanisms.

My Diary mimics the functionality of a typical blogging or journaling app. It includes user registration, login/logout, post creation, and content browsing. However, all of this is part of a controlled, sandboxed environment built to monitor suspicious behavior, analyze attack patterns, and improve intrusion detection systems.

The backend is instrumented to log user activity, generate alerts, and feed security analytics tools for further investigation. No actual sensitive or personal user data is stored or required. The entire setup is non-production and used exclusively in a closed, experimental network.

This project is part of a university thesis titled:
"Μελέτη τεχνικών για την ανίχνευση κακόβουλης εισβολής συστημάτων με χρήση εργαλείων ανοικτού λογισμικού"
(Study of intrusion detection in information systems using open-source tools)
